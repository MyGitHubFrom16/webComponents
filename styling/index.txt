:host - селектор хоста шедоу дома. Этот селектор имеет низкий приоритет, который может быть переписан из light dom

:host-context(.class) - если кто-либо из родителей хоста имеет класс .class, то стили которые пропишу в этом селекторе применятся, также сработает если этот класс повесить на хоста

:shadow - зааффектит только элемементы первого уровня в shadow tree (т.е. только shadow первого уровня)

:unresolved - псевдо селектор, чтобы спрятать кастомный элемент, пока не произойдет рендеринг, применяется только для кастомных элементов, которые еще не проинициализированы
.my-elem {opacity: 1;}
.my-elem:unresolved {opacity: 0;}
